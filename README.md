Четвёртый учебный проект. (https://mixachos.github.io/mesto/)

* HTML/CSS
* Grid вёрстка
* Адаптивная вёрстка
* Основы JS
* Работа с DOM
* Шаблоны разметки
* Валидация форм с помощью JS
* Обработка событий
* Основы ООП
* Модули JS
* Сборка webpack-ом
* Работа с API
